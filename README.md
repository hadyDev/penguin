# penguin
Engine base on opengl!

# How to install
```git clone https://github.com/hadyDev/penguin.git```

Please clone it on folder you want like Documents others.

# Create folder build in penguin folder
After that let`s build it!

```cmake ..```

Or you just need cofigure using cmake extensions on visual studio code.

```cmake --build .```

Wait until done!

Now run it.

```./engine```
